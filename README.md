Welcome to the Dealvago!

Dealvago is a project planned for the course Web-server programming. The main learning objective of this project is to learn and practice how to build a secure, scalable and dynamic web application using Spring Boot and Spring MVC.

It is a web-application focused on setting up proper web server integration and using Thymeleaf to dynamically generate basic HTML and plain text content from the server side. This project concentrates on backend development written in Java and to learn key concepts of sessions, concurrency, security and design patterns. 

The main building tool used is Maven and frameworks including

- Spring Boot
- Spring MVH
- Spring Data JPA
- Spring Security
- Spring Validation
- Thymeleaf

Other revisited topics include database modelling, SQL and MySQL.
